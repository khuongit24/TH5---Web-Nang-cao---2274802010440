**Cần thư viện và .env để chạy. Yêu cầu Node.js v18 trở lên**

Ví dụ .env:

MONGODB_URI=link_mongodb

PORT=3000

SESSION_SECRET=th5_secret_key_cua_khuong

Cài đặt thư viện: npm install

Khởi chạy: npm start


**Postman setup**: 

Chỉ cần import file TH5_Khuong_API.postman_collection.json vào app Postman.
